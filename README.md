# IBM-employee-churn-rate
The aim is to track the likelihood that employees would churn amongst the sample 1575 IBM. key factors such as years at the company, monthly income, percentage increase in salary had a large impact on the chance that an employee would leave or churn.
The visualisation libraries I primarily used was plotly and seaborn.
And for data preparation, model fitting and predictions I primarily used sckit learn.
The model that yielded the highest accuracy was the Random Forest Classifier (RF).
However, because it is by nature a divide and conquer algorithm, it is extremely slow
